Image Recreation - Genetic Algorithm

This is a genetic algorithm that I wrote in C for my 2S03 course (C Programming) at McMaster University. The purpose of the program is to try to recreate a given image starting from a random image with a random orientation of pixels. The video shown is a visual representation of the genetic algorithm as it trys to recreate the image for 100,000 iterations. It starts from a random image and with each iteration the image comes closer to resembling the image on the top of the video. Here is a link to the Github repo.

Technologies Used: C, BASH
