SmartJournal

SmartJournal is a web application that me and three other students created at HackWesternV, which was my first ever Hackathon! SmartJournal is a sentiment analysis tool that uses IBM Watson's tone analyzer API to analyze your written journal entry and determines your current mood. Based on the analysis, the application will then recommend nearby activities for you to do using the Google Maps API. The application also generates different quotes and words of consideration based on how you're feeling. You can try it out for yourself at helperjournal.com. Link to Github and Devpost.

Technologies Used:Javascript, HTML, CSS, stdlib, Node.js, Google-Maps, IBM Watson, Cloudflare
