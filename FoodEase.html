FoodEase

Have you ever been desperately hungry but didn't know what to cook for yourself? Well FoodEase aims to help you solve that exact problem! FoodEase is a web/mobile application that allows you to upload a picture of the ingredients you have in your fridge and recognizes the ingredients using Google's Cloud-Vision API. It then presents you with food recipes you can make with your available ingredients as well as the nutritional data associated with each recipe. I, along with three other students built this application for QHacks 2019 (my second Hackathon!). Here is a link to the Devpost and here is a link to the Github repo.

How we built it: Reading any .jpg files in the path defined, we load them into Google's Cloud-Vision-API giving us the ingredient name. From there we poll Edamam's API to give us the nutritional and Recipe information. Passing these in a JSON format to the Front-end where we pick it up using JavaScript. The Back-end is wrapped in Flask allowing for accessible ports and passing the information through.

Technolgies used: PHP, Django, Flask, MongoDB, SQL, Swift, Android Studio, Python, Node.js, JavaScript, HTML, CSS
